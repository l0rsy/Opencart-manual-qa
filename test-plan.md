# Test Plan — OpenCart Demo

## 1. Цель тестирования
Проверить основной функционал интернет-магазина OpenCart Demo
на соответствие ожидаемому поведению с точки зрения конечного пользователя.

## 2. Объект тестирования
- **Сайт:** https://demo.opencart.com
- **Версия:** OpenCart 4.x (демо-среда)
- **Тип:** Web-приложение (e-commerce)

## 3. Область тестирования (Scope)

### В области (In Scope)
- Регистрация и авторизация
- Каталог товаров и поиск
- Карточка товара
- Корзина
- Оформление заказа
- Личный кабинет

## 4. Виды тестирования
- Функциональное тестирование
- UI/UX тестирование
- Тестирование граничных значений
- Негативное тестирование

## 5. Критерии начала тестирования (Entry Criteria)
- Сайт доступен и открывается без ошибок
- Есть доступ к тестовой учётной записи или возможность регистрации
- Тест-кейсы написаны и согласованы

## 6. Критерии завершения тестирования (Exit Criteria)
- 100% тест-кейсов выполнено
- Все критические (Critical) и высокие (High) баги зафиксированы
- Нет открытых блокирующих дефектов

## 7. Тестовое окружение
| Параметр | Значение |
|----------|----------|
| ОС | Windows 10 |
| Браузер | Chrome (latest) |
| Разрешение | 1920x1080 |
| Инструменты | DevTools, GitHub, Markdown |
