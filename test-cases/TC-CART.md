# TC-CART — Корзина

## TC-010: Добавление товара в корзину

| Поле | Значение |
|------|----------|
| ID | TC-010 |
| Модуль | Cart |
| Тип | Позитивный |
| Приоритет | High |

**Предусловия:**
Открыта карточка любого товара. Qty (количество) = 1

**Шаги:**
1. Нажать кнопку **Add to Cart**

**Ожидаемый результат:**
Появляется уведомление "Success: You have added [товар] to your shopping cart!"
Счётчик корзины в шапке увеличивается на 1.

---

## TC-011: Изменение количества товара в корзине

| Поле | Значение |
|------|----------|
| ID | TC-011 |
| Модуль | Cart |
| Тип | Позитивный |
| Приоритет | Medium |

**Предусловия:**
В корзине есть минимум один товар. Открыта страница корзины `/index.php?route=checkout/cart`.

**Шаги:**
1. В поле Quantity изменить значение с `1` на `3`
2. Нажать кнопку **Update** (иконка обновления)

**Ожидаемый результат:**
Количество товара обновлено до 3.
Итоговая сумма пересчитана корректно.
