# TC-CART — Корзина

## TC-010: Добавление товара в корзину

| Поле | Значение |
|------|----------|
| ID | TC-010 |
| Модуль | Cart |
| Тип | Позитивный |
| Приоритет | High |

**Предусловия:**
Открыта карточка любого товара. Qty (количество) = 1

**Шаги:**
1. Нажать кнопку **Add to Cart**

**Ожидаемый результат:**
Появляется уведомление "Success: You have added [товар] to your shopping cart!"
Счётчик корзины в шапке увеличивается на 1.

---

## TC-011: Изменение количества товара в корзине

| Поле | Значение |
|------|----------|
| ID | TC-011 |
| Модуль | Cart |
| Тип | Позитивный |
| Приоритет | Medium |

**Предусловия:**
В корзине есть минимум один товар. Открыта страница корзины `/index.php?route=checkout/cart`.

**Шаги:**
1. В поле Quantity изменить значение с `1` на `3`
2. Нажать кнопку **Update** (иконка обновления)

**Ожидаемый результат:**
Количество товара обновлено до 3.
Итоговая сумма пересчитана корректно.

## TC-012: Удаление товара из корзины

| Поле | Значение |
|------|----------|
| ID | TC-012 |
| Модуль | Cart |
| Тип | Позитивный |
| Приоритет | High |

**Предусловия:**
В корзине есть минимум один товар. Открыта страница корзины.

**Шаги:**
1. Нажать кнопку **Remove** (иконка крестика) напротив товара

**Ожидаемый результат:**
Товар удалён из корзины.
Отображается сообщение "Your shopping cart is empty!" если товаров больше нет.

## TC-013: Переход в корзину с пустой корзиной

| Поле | Значение |
|------|----------|
| ID | TC-013 |
| Модуль | Cart |
| Тип | Негативный |
| Приоритет | Low |

**Предусловия:**
Корзина пуста.

**Шаги:**
1. Нажать на **Shopping Cart** иконку корзины в шапке сайта

**Ожидаемый результат:**
Открывается страница корзины с сообщением "Your shopping cart is empty!"
Кнопка Checkout отсутствует.
