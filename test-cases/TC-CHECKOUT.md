# TC-CHECKOUT — Оформление заказа

## TC-014: Переход к оформлению заказа авторизованным пользователем

| Поле | Значение |
|------|----------|
| ID | TC-014 |
| Модуль | Checkout |
| Тип | Позитивный |
| Приоритет | High |

**Предусловия:**
Пользователь авторизован. В корзине есть товар в наличии (например, HP LP3065).
**Шаги:**
1. Открыть корзину
2. Нажать кнопку **Checkout**

**Ожидаемый результат:**
Открывается страница оформления заказа `/index.php?route=checkout/checkout`.
Пользователь может заполнить данные доставки вручную.

## TC-016: Оформление заказа с валидными данными

| Поле | Значение |
|------|----------|
| ID | TC-016 |
| Модуль | Checkout |
| Тип | Позитивный |
| Приоритет | High |

**Предусловия:**
Пользователь авторизован. В корзине есть товар. Открыта страница checkout.

**Шаги:**
1. Заполнить адрес доставки валидными данными
2. Нажать **Continue**
3. Выбрать метод доставки `Flat Rate`
4. Выбрать метод оплаты `Cash On Delivery`
5. Нажать **Confirm Order**

**Ожидаемый результат:**
Заказ оформлен. Отображается страница подтверждения с надписью "Your order has been placed!"

## TC-017: Оформление заказа с незаполненным адресом

| Поле | Значение |
|------|----------|
| ID | TC-017 |
| Модуль | Checkout |
| Тип | Негативный |
| Приоритет | High |

**Предусловия:**
Пользователь авторизован. В корзине есть товар. Открыта страница checkout.

**Шаги:**
1. Оставить поле адреса пустым
2. Нажать **Continue** для перехода к следующему шагу

**Ожидаемый результат:**
Отображается сообщение об ошибке валидации "Address 1 must be between 3 and 128 characters!"
