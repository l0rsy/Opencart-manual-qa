# TC-AUTH — Авторизация и Регистрация

## TC-001: Успешная регистрация с валидными данными

| Поле | Значение |
|------|----------|
| ID | TC-001 |
| Модуль | Auth |
| Тип | Позитивный |
| Приоритет | High |

**Предусловия:**
Пользователь с таким Email не зарегистрирован. Открыта страница `/index.php?route=account/register`.

**Шаги:**
1. Заполнить First Name: `Test`
2. Заполнить Last Name: `User`
3. Заполнить Email: `testuser676767@gmail.com`
4. Заполнить Password: `Test1234!`
5. Активировать тумблер **I have read and agree to the Privacy Policy**
6. Нажать **Continue**

**Ожидаемый результат:**
Редирект на страницу подтверждения "Your Account Has Been Created!".
