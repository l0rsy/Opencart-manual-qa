# TC-ACCOUNT — Личный кабинет

## TC-018: Просмотр истории заказов

| Поле | Значение |
|------|----------|
| ID | TC-018 |
| Модуль | Account |
| Тип | Позитивный |
| Приоритет | Medium |

**Предусловия:**
Пользователь авторизован. Ранее был оформлен минимум один заказ.

**Шаги:**
1. Открыть личный кабинет
2. Перейти в раздел **Order History**

**Ожидаемый результат:**
Отображается список заказов с ID, Customer, кол-вом предметов, суммой, датой и кнопкой "View"

## TC-019: Редактирование данных аккаунта

| Поле | Значение |
|------|----------|
| ID | TC-019 |
| Модуль | Account |
| Тип | Позитивный |
| Приоритет | Medium |

**Предусловия:**
Пользователь авторизован. Открыт личный кабинет.

**Шаги:**
1. Перейти в **Edit your account information**
2. Изменить First Name на `Updated`
3. Нажать **Continue**

**Ожидаемый результат:**
Данные сохранены. Отображается сообщение " Success: Your account has been successfully updated."

## TC-020: Выход из аккаунта

| Поле | Значение |
|------|----------|
| ID | TC-020 |
| Модуль | Account |
| Тип | Позитивный |
| Приоритет | Medium |

**Предусловия:**
Пользователь авторизован.

**Шаги:**
1. В верхнем меню нажать на имя пользователя
2. Выбрать **Logout**

**Ожидаемый результат:**
Сессия завершена. Надпись "Account Logout
You have been logged off your account. It is now safe to leave the computer"
Доступ к личному кабинету без повторной авторизации недоступен.
